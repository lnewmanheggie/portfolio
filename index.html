About Me

Hello! My name is Lydia and I am a web developer based in Minneapolis, Minnesota. I am a recent graduate of the University of Minnesota coding bootcamp, with skills in JavaScript, React, Node.js, Express.js, REST APIs, MongoDB, and MySQL. I particularly enjoy the logic and functionality side of applications, and I am motivated to learn any new framework or technology.

Since my bootcamp, I have been working on a large scale development project for a plant sale database management system. My role in this project involves refactoring and writing code in PHP, JavaScript (jQuery to be specific), and HTML. We use Github for version control and follow an agile methodology. This plant sale helps Friends School of Minnesota (the school I went to as a kid) raise money for scholarships, and working on this project as a volunteer is a way for me to help the community and give back to the school I attended.

Check me out on Github and Linkedin!
